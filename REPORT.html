Security & Safety Features

Authentication & Authorization

  • JWT Sessions: 7-day expiration with refresh
  • Password Security: Bcrypt hashing with salting
  • Route Protection: verifySession() middleware on all protected routes
  • Role-Based Access: ADMIN, USER, and status-based checks
  • HttpOnly Cookies: Session tokens secured from XSS

Data Privacy & Protection

  • Profile Visibility: Blurred photos for non-members, contact details hidden
  • Blocking System: Complete user isolation when blocked
  • Data Deletion: Cascade deletion with automatic cleanup
  • GDPR Ready: Data structure supports compliance
  • Encryption: Sensitive data encrypted at rest

Input Validation & Injection Prevention

  • SQL Injection: Prisma ORM prevents all SQL injection
  • XSS Prevention: React automatic escaping, no dangerouslySetInnerHTML
  • CSRF Protection: Server-only form actions
  • Form Validation: Client-side (React) + Server-side (Zod)
  • File Upload Safety: Cloudinary validation + type restrictions

Safety Features

  • Block System: Users can block abusive profiles completely
  • Report System: 5 categorized report reasons for admin review
  • User Status Tracking: APPROVED, PENDING, REJECTED, SUSPENDED states
  • Admin Review Queue: Dashboard for managing reports
  • Verification Workflow: Email verification on signup
🔒 Security Note: The platform implements industry best practices for matrimonial sites. All external services (Stripe, Cloudinary, Resend) are PCI-DSS or equivalent compliant.
